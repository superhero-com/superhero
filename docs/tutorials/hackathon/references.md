- **[Firefox Extension](https://addons.mozilla.org/en-US/firefox/addon/superhero-wallet/)** - Install Superhero Wallet for Firefox

### Mobile Apps
- **[Android App](https://play.google.com/store/apps/details?id=com.superhero.cordova)** - Download Superhero Wallet for Android
- **[iOS App](https://apps.apple.com/us/app/superhero-wallet/id1502786641)** - Download Superhero Wallet for iOS

### Web Wallet

## Testnet Faucet

Get testnet AE tokens for development and testing:
- **[æternity Testnet Faucet](https://faucet.aepps.com/)** - Request testnet AE tokens

<Tip>
Switch your wallet to testnet in settings before requesting tokens from the faucet.
</Tip>
